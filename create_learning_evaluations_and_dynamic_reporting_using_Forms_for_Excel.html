Create learning evaluations and dynamic reporting using Forms for Excel


A. Introducing a brief overview of the Kirkpatrick Model

A. Introducing a brief overview of the Kirkpatrick Model
One of the most popular models for analyzing and evaluating the results of programs (training or educational) is the Kirkpatrick Model.



Level 1 Reaction: addresses how the participants react to the training session
Level 2 Learning: focuses on whether the attendees have understood the content
Level 3 Behavior: analyses the change in behavior. Are the participants utilizing what they have learned?
Level 4 Results: determines if the training had a positive impact on the organization/business

The questions in the evaluation form correlate directly to these levels.


B. Setting up Forms for Excel in order to store the evaluation responses
Here is a reference to the Microsoft Tech Community article introducing Forms for Excel.

The Excel contains each evaluation on one line. Imagine this as a repository, a database behind the evaluation form. Let's start by creating the Excel. Navigate to the location where the file should be created and clickÂ New and Forms for Excel.



Microsoft Forms automatically opens in a new tab.


C. Adding evaluation questions in Microsoft Forms
At this moment we can start to work on our evaluation form. There are a multitude of questions you can ask for each of the Kirkpatrick levels. The following are some of the (Rating) examples.
Subtitle - allows for adding extra information.

Level 1 - I found the session to be a good use of my time



Level 2 - I was appropriately challenged by the material



Level 3 - I will be able to apply what I have learned


For a more granular approach, consider the knowledge that the student should gain at the end of the session and, using a Likert entry, evaluate each skill.


Level 4 - This session will improve my performance and productivity


For a more granular understanding, using Multiple answers Choice, ask the student to reflect on which specific areas, the session had an impact on.


Net Promoter Score - I will recommend this session to others



Feedback - You can share below any additional comments or suggestions



Use the Preview button at the top of the form to submit a few responses.


D. Creating a dynamic report to present the results
Let's navigate to the location where the Excel was created (a different tab) and Open in app.
Form1 contains a few entries. You can notice the columns represent, amongst others, the questions from the Form. Rename the existing sheet to Calculations - it will be used to compute the averages across the responses.



How about the Net Promoter Score?



Next, let's visualize the results.
Create a New sheet, go to Page Layout and set the size to A4. Click View and select Page Layout. Insert a Doughnut Chart for each 4 levels with a rectangle in the middle containing the corresponding value. For the Net Promoter Score use rectangles to build a dashboard.



That is it! Test the dynamic changes in the report by submitting new evaluations.



You can now cleanup the responses, beautify the report and share the evaluation with your target audience.
